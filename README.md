# 🖨️ Panther Lab 3D

Site institucional desenvolvido para apresentar produtos e serviços de impressão 3D personalizados.

## 📋 Sobre o Projeto

A Panther Lab 3D é uma empresa focada na criação de produtos personalizados através da impressão 3D.

O site foi desenvolvido para apresentar os serviços, demonstrar trabalhos realizados e facilitar o contato com clientes interessados em peças exclusivas.

## ✨ Funcionalidades

- Página institucional moderna
- Apresentação dos serviços de impressão 3D
- Galeria de produtos personalizados
- Sessão de miniaturas e colecionáveis
- Exposição de mascotes personalizados
- Produtos temáticos de animes, filmes e esportes
- Área de contato integrada
- Layout totalmente responsivo

## 🎨 Produtos Destacados

- Miniaturas personalizadas
- Personagens de animes
- Personagens de filmes
- Caricaturas em 3D
- Chaveiros personalizados
- Suportes temáticos
- Porta-figurinhas personalizados
- Mascotes esportivos
- Peças decorativas exclusivas

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Design Responsivo
- Mobile First

## 📱 Responsividade

O site foi projetado para funcionar perfeitamente em:

- Smartphones
- Tablets
- Computadores

## 🎯 Objetivo

Apresentar os serviços da Panther Lab 3D de forma profissional, valorizando a qualidade dos produtos e facilitando a comunicação com potenciais clientes.
